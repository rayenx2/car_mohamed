# VIP Cars Images

This folder contains images for VIP/luxury cars that will be displayed in the masonry layout.

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 600x400px or similar aspect ratios
- File naming: vip-car-1.jpg, vip-car-2.jpg, etc.
