# Normal Cars Images

This folder contains images for regular/standard cars that can be used in other sections of the website.

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 600x400px or similar aspect ratios
- File naming: car-1.jpg, car-2.jpg, etc.

## Usage:
These images can be used for:
- Stock page car listings
- Featured cars sections
- Car gallery components
- Any other car-related displays

The images in this folder are separate from the VIP cars collection and can be used for different purposes throughout the website.
